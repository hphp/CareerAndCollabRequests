# 人群重复劳动与价值分布调研战略目标文档

> _版本：v1.0 | 日期：2026-04-10 | 编制：Eva_

---

## 1. 文档背景

### 1.1 核心命题

**分析中国的人群分布（以工种划分），以及每个人群中重复劳动可以被当前技术组合所替代的价值。**

$$
\text{可替代价值} = (\text{商业价值} + \text{个人定义价值}) \times \text{时间}
$$

- **商业价值**：该重复劳动为雇主/组织带来的经济收益
- **个人定义价值**：该重复劳动对从业者个人的意义（技能成长、成就感、社会关系等）
- **时间**：该重复劳动发生的频率和持续时长

### 1.2 战略意义

| 层面 | 价值 |
|------|------|
| **商业** | 识别具身智能 / AI 技术替代的高价值场景，量化市场规模 |
| **社会** | 预判技术替代对就业的影响，为政策制定提供数据支撑 |
| **技术** | 指明 AI + 机器人技术组合的落地优先方向 |
| **个人** | 帮助个体了解自身工作的可替代性，规划职业转型 |

---

## 2. 战略目标

### 2.1 总体目标

建立一套可量化、可复现的人群重复劳动与可替代价值评估体系，覆盖中国主要工种，产出「技术替代价值地图」。

### 2.2 分层目标

| 目标层级 | 描述 | 量化指标 |
|---------|------|---------|
| **O1：底座建设** | 建立工种分类体系和调研方法论 | 完成 ≥ 50 个核心工种的分类框架 |
| **O2：数据采集** | 完成全国范围的多渠道数据采集 | 有效样本 ≥ 10,000 份，覆盖 ≥ 30 个省份 |
| **O3：价值评估** | 完成每人群的重复劳动占比和可替代价值计算 | 每个工种输出：重复劳动占比(%)、年可替代价值(元) |
| **O4：地图输出** | 产出「中国人群可替代价值地图」可视化成果 | 交互式 Dashboard + 行业分析报告 |
| **O5：动态追踪** | 建立长期追踪机制，支持季度更新 | 季度复调机制上线，数据可对比基线 |

## 3. 调研内容框架

### 3.1 信息收集维度

#### 维度一：工种基础信息

| 信息项 | 说明 | 数据来源 |
|--------|------|---------|
| 工种名称 | 标准工种名称（如：流水线装配工、客服代表） | 国家职业分类大典 + 实地补充 |
| 行业归属 | 该工种所属的行业分类 | GB/T 4754 |
| 职业编码 | 国家标准职业编码 | 人社部 |
| 从业人数 | 全国估算从业人数 | 统计局、行业协会 |
| 地域分布 | 主要从业地区 | 招聘平台数据、统计年鉴 |
| 学历/技能要求 | 入门学历、必备技能 | 招聘网站 JD 分析 |
| 薪酬水平 | 平均月薪/年薪（含五险一金） | 智联、Boss直聘、统计局 |
| 工作环境 | 室内/室外/危险环境等 | 实地调研、行业报告 |

#### 维度二：工作任务拆解

| 信息项 | 说明 | 采集方式 |
|--------|------|---------|
| 任务清单 | 该工种涉及的所有具体任务 | 专家访谈 + 工人调研 |
| 任务频次 | 每项任务每天/周/月发生的次数 | 记工时调研 |
| 任务时长 | 每项任务平均单次耗时 | 记工时调研 |
| 任务重复度 | 任务的可标准化/可复制程度（1-10分） | 专家打分 + AI视频分析 |
| 任务自主性 | 任务执行中对判断/创造力的要求（1-10分） | 专家打分 |
| 任务人机协作现状 | 当前是否已有工具/机器辅助 | 问卷 + 实地走访 |

#### 维度三：重复劳动识别

| 信息项 | 说明 | 判定标准 |
|--------|------|---------|
| 重复劳动定义 | 有明确规则、可标准化、周期<1分钟的劳动行为 | 专家规则 + NLP判断 |
| 重复劳动占比 | 该工种中重复劳动占总工作时间的比例（%） | 记工时 + AI视频分析 |
| 重复劳动细分 | 重复劳动的具体类型：体力重复 / 认知重复 / 沟通重复 | 任务分类映射 |
| 替代技术组合 | 当前可实现替代的技术组合（AI大模型/机械臂/传感器等） | 技术专家评估 |
| 替代成熟度 | 技术替代的可行性：成熟/接近成熟/探索中 | 技术TRI分析 |

#### 维度四：价值评估

| 信息项 | 说明 | 计算方式 |
|--------|------|---------|
| **商业价值** | 该工种每单位重复劳动创造的直接经济价值 | 行业人均产出 ÷ 工作时长 × 重复占比 |
| **个人定义价值** | 从业者主观认为该工作的价值感/成长性/稳定性等（1-100分） | 问卷调研 |
| **年工作时数** | 该工种人均年工作时（小时） | 统计局 + 问卷校正 |
| **可替代价值总量** | 全国该工种可替代价值总量（元/年） | 从业人数 × 年工作时 × 重复占比 × 时均产出 |
| **替代成本** | 实现替代所需的硬件/软件/部署/培训总成本 | 厂商报价 + 实施估算 |

#### 维度五：社会影响评估

| 信息项 | 说明 | 数据来源 |
|--------|------|---------|
| 就业影响人数 | 若该工种30%被替代，受影响从业者数量 | 替代率模型推算 |
| 转型难度 | 该工种从业者转向其他岗位的平均难度（1-10分） | 专家评估 + 培训成本估算 |
| 政策敏感性 | 该工种是否涉及国家重点就业群体（1-10分） | 政策文件对照 |
| 替代时间窗口 | 预计该工种开始大规模被替代的时间（年） | 技术成熟度曲线 + 专家判断 |

## 4. 调研格式设计

### 4.1 定量问卷（从业者端）

#### 【问卷A：个人工作日志】

> 适用于大规模线上/线下采集

**A1. 基本信息**
| 字段 | 选项/格式 |
|------|----------|
| 年龄段 | 18-25 / 26-35 / 36-45 / 46-55 / 55+ |
| 性别 | 男 / 女 / 其他 |
| 学历 | 初中及以下 / 高中/职高 / 大专 / 本科 / 硕士及以上 |
| 工龄 | <1年 / 1-3年 / 3-5年 / 5-10年 / 10年+ |
| 所在行业 | （下拉：制造业/餐饮/客服/医疗/教育/…） |
| 岗位名称 | （文字输入） |
| 月收入区间 | <3000 / 3000-5000 / 5000-8000 / 8000-15000 / >15000 |
| 工作地点 | 省份 + 城市（文字输入） |

**A2. 工作任务拆解（记工时）**
> 指导语：请回忆您昨天（或上一个工作日）完成的所有工作任务，按时间顺序列出，并估计每项任务花费的时间。

| 任务# | 任务描述（请尽量详细描述您做了什么） | 花费时间（分钟） | 重复程度（1=完全不重复，10=完全重复） | 自主判断需求（1=完全不需要判断，10=完全依赖判断） |
|-------|------------------------------------|----------------|---------------------------------------|---------------------------------------------|

（请列出您当天所有任务，至少5项，至多15项）

**A3. 重复劳动自评**

| # | 问题 | 选项 |
|---|------|------|
| B1 | 您工作中有多少比例的内容是"每天几乎相同、重复进行的"？ | <10% / 10-30% / 30-50% / 50-70% / >70% |
| B2 | 您工作中哪些重复任务最消耗时间？（可多选） | 体力操作 / 数据录入/整理 / 沟通回复 / 巡查检查 / 搬运/移动 / 其他：___ |
| B3 | 您认为如果把重复工作交给机器/AI，您的工作会变成什么样？ | 更有价值 / 没变化 / 会失业 / 不确定 |
| B4 | 您目前工作中使用了哪些工具/机器/软件来辅助？ | （文字输入） |

**A4. 个人价值感知**

| # | 问题 | 评分（1-10） |
|---|------|------------|
| C1 | 这份工作给您带来的收入稳定性 | （1-10） |
| C2 | 这份工作让您感到有成就感的程度 | （1-10） |
| C3 | 这份工作让您感到技能成长的程度 | （1-10） |
| C4 | 如果重来一次，您还会选择这份工作吗？ | 一定会 / 大概率会 / 不确定 / 大概率不会 / 一定不会 |

### 4.2 专家访谈大纲

#### 【访谈D：行业专家/雇主端】

**D1. 行业概况**
- 该行业/工种当前的整体发展现状如何？
- 近3年有哪些技术应用趋势？自动化/AI替代进展到了什么程度？
- 行业内对"重复劳动"的定义和边界在哪里？

**D2. 任务拆解**
- 这个工种最耗时的重复性任务是什么？（按时间排名前5）
- 这些重复任务的标准化程度如何？哪些规则可以固化到机器/AI中？
- 目前有没有尝试过用技术替代这些任务？效果如何？

**D3. 成本收益**
- 这个工种的年人力成本（含工资/保险/管理）大约是多少？
- 如果引入自动化/AI方案，预期的投入成本和回收周期？
- 您认为替代的最大障碍是什么？（技术/成本/人员/监管？）

**D4. 替代预测**
- 您判断这个工种在3年/5年/10年内被部分替代的可能性有多大？
- 替代后对剩下的人力有什么新要求？

## 6. 价值评估模型

### 6.1 核心公式

$$
V_{\text{replaceable}} = \sum_{j=1}^{J} \left( N_j \times H_j \times R_j \times \left( \frac{S_j}{H_j} \right) \right)
$$

其中：
- $V_{\text{replaceable}}$：全国该工种年可替代价值总量（元/年）
- $N_j$：第 $j$ 类任务的重复劳动占比（0~1）
- $H_j$：人均年工作时（小时）
- $R_j$：该工种全国从业人数
- $S_j$：该工种人均年产出/薪酬（元/年）

### 6.2 简化估算公式

$$
\text{可替代价值总量} = \text{从业人数} \times \text{年人力成本} \times \text{重复劳动占比}
$$

### 6.3 替代ROI计算

$$
\text{替代ROI} = \frac{\text{年度可替代价值}}{\text{替代总投入成本（含硬件/软件/部署/培训）}}
$$

| ROI区间 | 替代优先级 |
|--------|-----------|
| >3x | 🔴 极高优先级：立刻投入 |
| 2x-3x | 🟠 高优先级：半年内启动 |
| 1x-2x | 🟡 中优先级：1-2年内观察 |
| <1x | 🟢 低优先级：等待技术成本下降 |

## 8. 质量控制

### 8.1 数据质量校验

| 校验规则 | 说明 |
|---------|------|
| 任务时长一致性 | 单个工作日任务总时长应在4-12小时范围内，否则标记 |
| 重复度-自主性负相关 | 重复度高(>7)但自主性也高(>7)的样本需复核 |
| 异常值过滤 | 薪酬超出行业均值5倍以上的样本需复核 |
| 交叉验证 | 问卷数据与招聘平台JD数据做交叉校验 |

### 8.2 专家评审机制

- 每个重点工种（10个）至少邀请2位行业专家进行数据评审
- 建立「调研质量评分卡」，对数据可信度分级

## 10. 风险与应对

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|---------|
| 问卷回收率低 | 中 | 高 | 多渠道并行 + 激励加强 + 企业合作 |
| 数据失真（受访者低报重复劳动） | 中 | 中 | AI视频分析交叉验证 + 专家评审 |
| 工种分类边界模糊 | 中 | 中 | 建立「典型任务-工种」映射矩阵，允许重叠 |
| 敏感行业数据获取困难 | 高 | 中 | 优先从公开数据入手，敏感行业用估算模型 |
| 技术快速迭代导致预测失效 | 高 | 低 | 建立动态更新机制，预测仅作参考 |

| 术语 | 定义 |
